  Détection de DeepFakes avec XceptionNet

  Description
Implémentation d'un classificateur de deepfakes basé sur XceptionNet fine-tuné sur FaceForensics++. 
Le modèle atteint **81.13% d'accuracy** pour distinguer images originales et deepfakes.

  Installation Rapide

 Prérequis
- Python 3.8+
- PyTorch 2.0+
- CUDA 11.6+ (recommandé pour GPU)

 Installation
dans ton bash :
 1. Cloner le dépôt
git clone https://github.com/issamentacode/deepfake-detection-project-XceptionNet.git
cd deepfake-detection-project-XceptionNet

 2. Installer les dépendances
pip install -r requirements.txt

 3. Télécharger le dataset (dans la section Dataset)
